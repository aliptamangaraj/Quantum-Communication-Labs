# Theory

Quantum teleportation is one of the fundamental protocols in quantum information science. Contrary to its name, quantum teleportation does not involve the physical transfer of a particle. Instead, it transfers the quantum state of one qubit to another distant qubit using:

Quantum entanglement
Classical communication
Local quantum operations

The protocol was first proposed by Bennett et al. (1993).

### Participants
**Alice** possesses: Unknown quantum state; One qubit of an entangled Bell pair
**Bob** possesses: The second qubit of the Bell pair

Alice performs

1. CNOT
2. Hadamard

on her two qubits.

She then measures both qubits.

### Classical Communication

Alice sends the two measurement bits to Bob using a classical communication channel.

Depending on the received bits, Bob performs correction operations.

Alice's Measurement	  Bob's Operation
00	                     Identity
01	                        X
10	                        Z
11	                        ZX

After these corrections, Bob's qubit becomes identical to the original unknown state.

# Qiskit Implementation

## Quantum Teleportation with Measurements

### Steps:
1. Import required libraries.
2. Define an arbitrary quantum state.
3. Initialize qubit 0.
4. Generate Bell pair.
5. Perform Bell measurement.
6. Measure Alice's qubits.
7. Apply conditional X and Z corrections.
8. Measure all qubits.

### Code:

from qiskit import QuantumCircuit
from qiskit.quantum_info import (
    Statevector,
    DensityMatrix,
    partial_trace,
    state_fidelity
)

from qiskit_aer import AerSimulator

import numpy as np

alpha = np.sqrt(0.3)
beta = np.sqrt(0.7)

psi = Statevector([alpha, beta])

print(psi)

qc = QuantumCircuit(3,2)

# Unknown state
qc.initialize(psi.data,0)

# Bell Pair
qc.h(1)
qc.cx(1,2)

# Bell Measurement Operations
qc.cx(0,1)
qc.h(0)

qc.measure(0, 0) 
qc.measure(1, 1)

with qc.if_test((1, 1)):
qc.x(2)

with qc.if_test((0, 1)):
qc.z(2)
qc.barrier()

qc.measure_all()
qc.draw()

## Fidelity Verification

### The second program:

from qiskit import QuantumCircuit
from qiskit.quantum_info import (
    Statevector,
    DensityMatrix,
    partial_trace,
    state_fidelity
)
import numpy as np

# ----------------------------------------------------
# Prepare an arbitrary state
# ----------------------------------------------------

alpha = np.sqrt(0.3)
beta = np.sqrt(0.7)

psi = Statevector([alpha, beta])

print("Original State")
print(psi)
print()

# ----------------------------------------------------
# Teleportation circuit (without measurements)
# ----------------------------------------------------

qc = QuantumCircuit(3)

# Unknown state on qubit 0
qc.initialize(psi.data, 0)

# Bell pair between qubit 1 and 2
qc.h(1)
qc.cx(1, 2)

# Bell measurement operations
qc.cx(0, 1)
qc.h(0)

# State after Bell measurement operations
state = Statevector.from_instruction(qc)

print("Three-Qubit State")
print(state)
print()

# ----------------------------------------------------
# Project onto measurement outcome |00>
# ----------------------------------------------------

# Amplitudes corresponding to Alice = |00>
# Basis ordering:
# |000>,|001>,|010>,|011>,|100>,|101>,|110>,|111>

amps = np.array([
    state.data[0],
    state.data[1]
])

# Normalize Bob's state
amps = amps / np.linalg.norm(amps)

bob_state = Statevector(amps)

print("Bob State (Measurement Outcome = 00)")
print(bob_state)
print()

# ----------------------------------------------------
# Density Matrix
# ----------------------------------------------------

bob_density = DensityMatrix(bob_state)

print("Bob Density Matrix")
print(bob_density)
print()

# ----------------------------------------------------
# Fidelity
# ----------------------------------------------------

fidelity = state_fidelity(psi, bob_density)

print("Teleportation Fidelity")
print(fidelity)


1. prepares the same unknown state,
2. performs teleportation without measurements,
3. extracts Bob's state,
4. constructs the density matrix,
5. computes the teleportation fidelity.

This experiment successfully demonstrated the quantum teleportation protocol using Qiskit.

An arbitrary quantum state was prepared and teleported from Alice's qubit to Bob's qubit through a shared Bell pair and classical communication.
